 
# Artisan Marketplace

## Overview
Artisan Marketplace is a web-based platform connecting artisans directly with customers. It allows artisans to register, submit handmade or hobby-based products, and sell online. Admin approval ensures quality and authenticity, while role-based access control secures the platform.

## Features
- User registration & login (Artisan / Customer / Admin)  
- Role-based access control  
- Artisan product submission with admin approval  
- Product listing with images, prices, and descriptions  
- Customer browsing & purchase of approved products  
- Order management system  
- Admin dashboard for reviewing and approving products  
- Responsive UI using Bootstrap  
- Scalable backend with Django and Google Cloud  

## Technologies
- **Backend:** Django  
- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Database:** SQlite  
- **Version Control:** GitHub  
- **AI Assistance:** ChatGPT for code support  

## Setup
1. Clone repo: `git clone https://github.com/SHRAMAJEEVI/Artisan_Market_place.git`  
2. Go to directory: `cd Artisan_Market_place`  
3. Create & activate virtual environment:  
   - Linux/Mac: `python -m venv env && source env/bin/activate`  
   - Windows: `python -m venv env && env\Scripts\activate`  
4. Install dependencies: `pip install -r requirements.txt`  
5. Run migrations: `python manage.py migrate`  
6. Start server: `python manage.py runserver`  
7. Open: `http://127.0.0.1:8000/`  

## Usage
- **Artisan:** Submit products, view own list (pending admin approval)  
- **Admin:** Approve or reject products  
- **Customer:** Browse & purchase approved products
